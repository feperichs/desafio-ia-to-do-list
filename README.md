## 📌 Sobre o Projeto
Este projeto atende aos requisitos do desafio: Campo de entrada, seletor de prioridade (Alta/Média/Baixa), lista dinâmica e ação de conclusão com estilização condicional via CSS.

## 🧠 Reflexão Crítica (15% da Nota)
- **Geração Consciente:** O código JavaScript foi escrito linha a linha com entendimento do DOM. A IA foi utilizada para sugerir a estrutura de `flex` e as variáveis CSS, mas a lógica de `updateCounter()` e a manipulação do `checkbox:checked + .task-text` foi validada manualmente.
- **Engenharia de Prompt:** Os prompts seguiram os pilares: Persona (Especialista em CSS), Contexto (To-Do List sem frameworks), Tarefa (Estilizar prioridades) e Formato (Apenas código).
- **Acessibilidade:** Foram incluídos `aria-label` e foco visível (`focus-visible`).
